Current Project: ML Based Prediction Server for X Peptides

A prediction server is being developed using a machine learning classification model, utilizing the same peptide dataset created for master thesis. The positive data set consisted of peptides from the developed database, while the negative data set was created using the Python Random library. Feature calculation and top-performing feature selection will be performed followed by model training.

A user-friendly webserver interface will be developed for the prediction server.

Master's Thesis: A Comprehensive Database on X Peptides

Bioinformatic analysis of amino acid sequences of peptides/proteins with specific activity collected from PubMed literature and Lens patents was performed. Using the latest structural biology tools and deep learning methods, determination of the physicochemical, structural, and motif properties of the peptides/proteins were carried out.

A user-friendly web server interface was created for the database, allowing easy navigation, visual representation, download options, and access to contact information.

Summer Internship 2022: Exosomal Gene Expression Analysis on Pancreatic Cancer RNA-Seq Data

From the RNA-seq data of pancreatic cancer (177 samples) from cBioPortal and the normal RNA sequencing data (30 samples) from GSE, exosomal gene expression analysis was performed. Exosome genes (100 genes) were obtained from the ExoCarta database. The goal was to see which exosomal genes were expressed differently in pancreatic cancer.

Batch effects were addressed using the PyCombat library, which was then validated by cluster analysis. The up-regulation and down-regulation of exosomal genes were visualised using heat maps and box plots after statistical analysis.
